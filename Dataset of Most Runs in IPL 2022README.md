
# Dataset of Most Runs in IPL 2022
This dataset contains information about the top run scorers in the Indian Premier League (IPL) 2022 season. The data was collected from official sources and manually curated.

# Data Description
#### The dataset contains the following columns:

Player: The name of the player.

Team: The name of the player's team.

Matches: The number of matches played by the player.

Innings: The number of innings played by the player.

Not_Out: The number of times the player was not out.

Runs: The total number of runs scored by the player.

Highest_Score: The highest individual score achieved by the player.

Average: The batting average of the player.

Ball_Faced: The total number of balls faced by the player.

Strike_Rate: The strike rate of the player.

100s: The number of centuries scored by the player.

50s: The number of half-centuries scored by the player.

4s: The total number of 4s scored by the player.

6s: The total number of 6s scored by the player.

# Data Source
The data was collected from official sources such as IPL's website and manually curated to ensure accuracy.

# Usage
This dataset can be used for various purposes such as analysis, visualization, and machine learning. However, please cite the source of the data if you use it in your work.
